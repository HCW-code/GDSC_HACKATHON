# GDSC_HACKATHON
일상의 역발상(흔치않은 듣기 좋은 음악 추천 웹)

## 개발환경
* Node.js v16
* Npm v8.3.1
* Iframe
* MySQL workbench

1. 홈화면에서는 흔하게 들을 수 없지만 사용자들의 다시듣기 횟수가 높은 음악리스트를 확인하고 들을 수 있습니다.
